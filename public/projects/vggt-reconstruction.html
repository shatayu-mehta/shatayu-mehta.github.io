Dual Segmentation Approach

Method A: YOLO+SAM Detection - Uses YOLO-World for open-vocabulary object detection followed by SAM for precise mask generation. Ideal for well-textured objects with clear boundaries.

Method B: Geometric Propagation - Detects objects in a reference frame and propagates masks across frames using homography and depth information. More robust for texture-less objects.
